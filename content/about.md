## Hi, I'm Walter

I'm a DevOps Engineer based in Italy with a passion for building and maintaining cloud infrastructure at scale. My focus is on automation, reliability engineering, and helping teams ship software faster and more safely.

I believe that good infrastructure should be invisible — when everything works smoothly, teams can focus on what matters most: delivering value to users.

## What I Do

- **Kubernetes**: Building and operating production Kubernetes clusters, from bare-metal to managed cloud solutions
- **Infrastructure as Code**: Terraform and Ansible for reproducible, version-controlled infrastructure
- **CI/CD Pipelines**: GitHub Actions, GitLab CI, and FluxCD for GitOps workflows
- **Observability**: Prometheus, Grafana and using OpenTelemetry to keep systems healthy
- **Cloud Platforms**: Experience with Azure, Oracle Cloud and on-premise environments
- **LLM / AI**: Exploring large language models, local inference, and AI tooling integration

## My Approach

I'm a strong advocate for:

- **Automation First**: If you're doing it twice, automate it
- **Infrastructure as Code**: Everything should be versioned, reviewed, and reproducible
- **Continuous Improvement**: Small, incremental changes over big-bang deployments
- **Documentation**: Good docs are as important as good code
- **Collaboration**: Breaking down silos between development and operations

I'm always happy to connect with fellow engineers and discuss DevOps, cloud architecture, or anything tech-related.
